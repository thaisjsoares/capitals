## Sobre o projeto

Comecei esse projeto para estudar Vue. O jogo usa a API [Rest Countries](https://restcountries.com/) para mostrar os países e suas capitais. 

## Tecnologias

- Vue 3

## Rodando o projeto
```
npm install
```
```
npm run serve
```
